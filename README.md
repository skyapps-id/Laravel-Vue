# Laravel 8 + Vue

### Dependencies :
  - Jwt-auth
  - Vuetify


### Feature Uses :
- Migration
- Seeder
- Eloquent
- Blade
- Middleware
- FormRequest
- Queue

### Start
```sh
$ composer install
$ configuration .env
$ php artisan migrate
$ php artisan db:seed
$ php artisan jwt:secret
$ yarn . or npm install (recommended using yarn)
$ php artisan serve
$ yarn run dev (development)
```

### End Point
- http://localhost:8000/
- http://localhost:8000/register/event/{uuid}
